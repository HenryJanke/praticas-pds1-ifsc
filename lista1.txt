1 - Em programação, o que é uma biblioteca?
R: Em programação biblioteca é uma maneira de compilar códigos de maneira automática, semelhante a uma função subentendida no programa para facilitar ou adicionar novas 
funcionalidades no programa onde programando manualmente demoraria muito.

2 - O que é a biblioteca Swing do Java e qual é o seu propósito?
R: O Swing do Java é uma biblioteca com o objetivo de criar interfaces gráficas.
